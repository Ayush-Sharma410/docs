Plugin 은 Protobuf 를 사용하는 어떤 언어로도 개발 가능하다.  
Micro Service 와 Plugin 통신은 모두 Protobuf 를 기본으로 사용하기 때문이다. 기본적인 구조는 gRPC interface 를 사용한 서버 개발 과정과 동일한 방식이다.
